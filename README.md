# **InstantCheck**

Monitoramento do usuário temperatura, pressão e outros, enviar um alerta para o usuário indicando as variações

 ## **Build**

Baixar os arquivos do repositório  e rodar os seguintes comandos.
- yarn
- yarn android

## **Requisitos**

Configurar o ambiente seguindo o passo a passo do React Native CLI Quickstart no  link abaixo:

https://reactnative.dev/docs/environment-setup

Node:

https://nodejs.org/en/

Yarn:

https://classic.yarnpkg.com/pt-BR/docs/install/#windows-stable


Git:

https://git-scm.com/download/win
